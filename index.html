Claude responded: .

.env.local — это файл с секретными ключами

Простыми словами: это файл где хранятся пароли и ключи твоего проекта. Он никогда не уходит на GitHub — только на твоём компьютере и на Vercel.

Зачем он нужен?

Чтобы AI-ментор мог обращаться к Claude, ему нужен API ключ от Anthropic — это как пароль для доступа к AI. Этот ключ нельзя вставлять прямо в код (утечёт на GitHub), поэтому он хранится в .env.local.

Как создать — 3 шага

Шаг 1 — Получи ключ: Зайди на console.anthropic.com → API Keys → Create Key → скопируй

Шаг 2 — Создай файл .env.local в корне проекта:

vektorabo/
├── app/
├── components/
├── .env.local   ← вот сюда

Шаг 3 — Вставь внутрь одну строку:

ANTHROPIC_API_KEY=sk-ant-api03-ТВОЙ_КЛЮЧ_ЗДЕСЬ

На Vercel отдельно

Файл .env.local не едет на GitHub, поэтому на Vercel нужно добавить ключ вручную:

Vercel → твой проект → Settings → Environment Variables → добавь:

  • Name: ANTHROPIC_API_KEY
  • Value: sk-ant-api03-...
